git-crypt - Git дёӯзҡ„йҖҸжҳҺж–Үд»¶еҠ еҜҶ вҖӢ

git-crypt дҪҝ Git д»“еә“дёӯзҡ„ж–Үд»¶еҸҜд»Ҙе®һзҺ°йҖҸжҳҺзҡ„еҠ еҜҶе’Ңи§ЈеҜҶгҖӮжӮЁеҸҜд»ҘйҖүжӢ©йңҖиҰҒдҝқжҠӨзҡ„ж–Үд»¶пјҢеңЁжҸҗдәӨж—¶иҝӣиЎҢеҠ еҜҶпјҢеңЁжЈҖеҮәж—¶иҝӣиЎҢи§ЈеҜҶгҖӮgit-crypt еҸҜд»Ҙи®©жӮЁиҮӘз”ұе…ұдә«еҢ…еҗ«е…¬е…ұе’Ңз§ҒжңүеҶ…е®№зҡ„д»“еә“гҖӮеҚідҪҝејҖеҸ‘иҖ…жІЎжңүеҜҶй’ҘпјҢgit-crypt д№ҹиғҪи®©е…¶е®һзҺ°е…ӢйҡҶе’ҢжҸҗдәӨеҗ«жңүеҠ еҜҶж–Үд»¶зҡ„д»“еә“гҖӮиҝҷж ·пјҢжӮЁеҸҜд»Ҙе°Ҷз§ҳеҜҶиө„ж–ҷпјҲеҰӮеҜҶй’ҘжҲ–еҜҶз ҒпјүеӯҳеӮЁеңЁдёҺд»Јз ҒзӣёеҗҢзҡ„д»“еә“дёӯпјҢиҖҢдёҚйңҖиҰҒй”Ғе®ҡж•ҙдёӘд»“еә“гҖӮ

git-crypt жҳҜз”ұ Andrew AyerпјҲ agwa@andrewayer.name пјүзј–еҶҷзҡ„гҖӮ жӣҙеӨҡдҝЎжҒҜпјҢиҜ·и®ҝй—® https://www.agwa.name/projects/git-cryptгҖӮ

.gitattributes ж–Үд»¶и·ҹ .gitignore зұ»дјјпјҢеҸҜд»ҘеҢ№й…ҚйҖҡй…Қз¬ҰпјҢе№¶дё”еә”иҜҘжҸҗдәӨеҲ°д»“еә“йҮҢгҖӮжӣҙеӨҡдҝЎжҒҜеҸҜд»ҘжҹҘзңӢдёӢж–№еҶ…е®№гҖӮиҜ·зЎ®и®ӨдёҚиҰҒй”ҷиҜҜең°еҠ еҜҶдәҶ .gitattributes ж–Үд»¶жң¬иә«пјҲжҲ–иҖ…е…¶д»– Git ж–Үд»¶пјҢдҫӢеҰӮ .gitignore жҲ–иҖ… .gitmodulesпјүгҖӮиҜ·зЎ®и®ӨдҪ зҡ„ .gitattributes и§„еҲҷе·Із»Ҹз”ҹж•ҲеңЁжҸҗдәӨж•Ҹж„ҹж–Үд»¶д№ӢеүҚпјҢеҗҰеҲҷиҝҷдәӣж–Үд»¶е°ҶдёҚдјҡиў«еҠ еҜҶпјҒ

вҖӢ USER_ID вҖӢ еҸҜд»ҘжҳҜдёҖдёӘеҜҶй’Ҙ IDгҖҒе®Ңж•ҙжҢҮзә№гҖҒз”өеӯҗйӮ®д»¶ең°еқҖжҲ– е…¶д»–д»»дҪ•е”ҜдёҖж ҮиҜҶ GPG е…¬й’Ҙзҡ„ж–№ејҸпјҲеҸӮи§Ғ gpg жүӢеҶҢйЎөдёӯзҡ„ "HOW TO SPECIFY A USER ID"пјүгҖӮжіЁж„Ҹпјҡgit-crypt add-gpg-user вҖӢ дјҡ еңЁд»“еә“ж №зӣ®еҪ•зҡ„ .git-crypt зӣ®еҪ•дёӢж·»еҠ е№¶жҸҗдәӨдёҖдёӘ GPG еҠ еҜҶзҡ„еҜҶй’Ҙж–Үд»¶гҖӮ

иҝҷе°ұжҳҜдҪ йңҖиҰҒеҒҡзҡ„е…ЁйғЁе·ҘдҪңвҖ”вҖ”еңЁ git-crypt вҖӢ й…ҚзҪ®е®ҢжҲҗеҗҺпјҲж— и®әжҳҜйҖҡиҝҮ git-crypt init вҖӢ иҝҳжҳҜ git-crypt unlock вҖӢпјүпјҢдҪ еҸҜд»ҘеғҸеҫҖеёёдёҖж ·дҪҝз”Ё git вҖӢгҖӮеҠ еҜҶе’Ңи§ЈеҜҶиҝҮзЁӢдјҡйҖҸжҳҺең°иҝӣиЎҢгҖӮ

еҪ“еүҚзҠ¶жҖҒ вҖӢ

git-crypt зҡ„жңҖж–°зүҲжң¬жҳҜ 0.7.0пјҢеҸ‘еёғдәҺ 2022-04-21гҖӮgit-crypt ж—ЁеңЁе®һзҺ°ж— зјәйҷ·дё”еҸҜйқ пјҢеҚіе®ғдёҚеә”еҙ©жәғгҖҒеӨұж•ҲжҲ–жі„йңІжӮЁзҡ„жңәеҜҶж•°жҚ®гҖӮз„¶иҖҢпјҢе®ғе°ҡжңӘжҲҗзҶҹпјҢеҚіе®ғзҡ„ж–ҮжЎЈгҖҒеҠҹиғҪдё°еҜҢжҖ§жҲ–жҳ“з”ЁжҖ§е°ҡжңӘиҫҫеҲ°еә”жңүзҡ„ж°ҙе№ігҖӮжӯӨеӨ–пјҢеңЁ 1.0 зүҲжң¬д№ӢеүҚпјҢеҸҜиғҪдјҡеј•е…ҘдёҚеҗ‘еҗҺе…је®№зҡ„еҸҳжӣҙгҖӮ

е®үе…ЁжҖ§ вҖӢ

git-crypt жҜ”е…¶д»–йҖҸжҳҺзҡ„ git еҠ еҜҶзі»з»ҹжӣҙе®үе…ЁгҖӮ git-crypt дҪҝз”Ё AES-256 еңЁ CTR жЁЎејҸдёӢеҠ еҜҶж–Үд»¶пјҢе№¶дҪҝз”Ёд»Һж–Үд»¶зҡ„ SHA-1 HMAC жҙҫз”ҹзҡ„еҗҲжҲҗ IVгҖӮ иҝҷз§Қж“ҚдҪңжЁЎејҸеңЁзЎ®е®ҡжҖ§йҖүжӢ©жҳҺж–Үж”»еҮ»дёӢиў«иҜҒжҳҺжҳҜиҜӯд№үе®үе…Ёзҡ„гҖӮ иҝҷж„Ҹе‘ізқҖе°Ҫз®ЎеҠ еҜҶжҳҜзЎ®е®ҡжҖ§зҡ„пјҲиҝҷжҳҜеҝ…йңҖзҡ„пјҢд»Ҙдҫҝ git еҸҜд»ҘеҢәеҲҶж–Үд»¶жҳҜеҗҰе·Іжӣҙж”№пјүпјҢ дҪҶе®ғдёҚдјҡжі„йңІд»»дҪ•дҝЎжҒҜпјҢеҸӘдјҡжҡҙйңІдёӨдёӘж–Үд»¶жҳҜеҗҰзӣёеҗҢгҖӮ е…¶д»–йҖҸжҳҺзҡ„ git еҠ еҜҶж–№жЎҲдҪҝз”Ё ECB жҲ– CBC жЁЎејҸе’Ңеӣәе®ҡ IVгҖӮиҝҷдәӣзі»з»ҹеңЁиҜӯд№үдёҠдёҚе®үе…ЁпјҢдјҡжі„йңІдҝЎжҒҜгҖӮ

еұҖйҷҗжҖ§ вҖӢ

git-crypt дҫқиө–дәҺ git иҝҮж»ӨеҷЁпјҢиҖҢиҝҷдәӣиҝҮж»ӨеҷЁе№¶йқһдёәеҠ еҜҶи®ҫи®ЎгҖӮеӣ жӯӨпјҢgit-crypt е№¶йқһеҠ еҜҶзүҲжң¬еә“дёӯеӨ§еӨҡж•°жҲ–е…ЁйғЁж–Үд»¶зҡ„жңҖдҪіе·Ҙе…·гҖӮgit-crypt зңҹжӯЈж“…й•ҝзҡ„жҳҜиҝҷж ·дёҖз§ҚеңәжҷҜпјҡдҪ зҡ„зүҲжң¬еә“з»қеӨ§еӨҡж•°еҶ…е®№жҳҜе…¬ејҖзҡ„пјҢдҪҶе…¶дёӯжңүдёҖдәӣж–Үд»¶пјҲеҸҜиғҪжҳҜдёҖз»„е‘ҪеҗҚдёә*. key зҡ„з§Ғй’ҘпјҢжҲ–жҳҜдёҖдёӘеҗ«жңү API еҮӯиҜҒзҡ„ж–Үд»¶пјүйңҖиҰҒеҠ еҜҶгҖӮеҰӮжһңйңҖиҰҒеҠ еҜҶж•ҙдёӘзүҲжң¬еә“пјҢиҖғиҷ‘дҪҝз”Ё git-remote-gcrypt иҝҷзұ»е·Ҙе…·гҖӮпјҲжіЁпјҡжң¬ж–ҮдёҚеҜ№ git-remote-gcrypt зҡ„е®үе…ЁжҖ§дҪңеҮәиҜ„д»·гҖӮпјү

git-crypt дёҚдјҡеҠ еҜҶж–Үд»¶еҗҚгҖҒжҸҗдәӨдҝЎжҒҜгҖҒз¬ҰеҸ·й“ҫжҺҘзӣ®ж ҮгҖҒgitlink жҲ–е…¶д»–е…ғж•°жҚ®гҖӮ

git-crypt ж— жі•йҡҗи—Ҹж–Үд»¶жҳҜеҗҰеҸ‘з”ҹжӣҙж”№гҖҒж–Үд»¶й•ҝеәҰпјҢжҲ–иҖ…дёӨдёӘж–Үд»¶е®Ңе…ЁзӣёеҗҢиҝҷдёҖдәӢе®һпјҲеҸӮи§ҒдёҠж–№вҖңе®үе…ЁжҖ§вҖқйғЁеҲҶпјү

git-crypt е№¶дёҚж”ҜжҢҒж’Өй”ҖеҜ№д№ӢеүҚе·ІжҺҲдәҲи®ҝй—®жқғйҷҗзҡ„еҠ еҜҶд»“еә“зҡ„и®ҝй—®гҖӮиҝҷж—ўйҖӮз”ЁдәҺеӨҡз”ЁжҲ· GPG жЁЎејҸпјҲжІЎжңү del-gpg-user е‘Ҫд»ӨжқҘиЎҘе…… add-gpg-userпјүпјҢд№ҹйҖӮз”ЁдәҺеҜ№з§°еҜҶй’ҘжЁЎејҸпјҲдёҚж”ҜжҢҒиҪ®жҚўеҜҶй’ҘпјүгҖӮиҝҷжҳҜеӣ дёәпјҢеңЁеҺҶеҸІж•°жҚ®зҡ„иғҢжҷҜдёӢпјҢиҝҷжҳҜдёҖдёӘеӣәжңүзҡ„еӨҚжқӮй—®йўҳгҖӮдҫӢеҰӮпјҢеҚідҪҝжҹҗдёӘж—¶еҲ»еҜҶй’Ҙиў«иҪ®жҚўдәҶпјҢжӢҘжңүд№ӢеүҚеҜҶй’Ҙзҡ„з”ЁжҲ·д»Қз„¶еҸҜд»Ҙи®ҝй—®д№ӢеүҚзҡ„д»“еә“еҺҶеҸІгҖӮиҝҷдёӘй—®йўҳеңЁ https://github.com/AGWA/git-crypt/issues/47 дёӯжңүжӣҙиҜҰз»Ҷзҡ„и®Ёи®әгҖӮ

дҪҝз”Ё git-crypt еҠ еҜҶзҡ„ж–Үд»¶ж— жі•иў«еҺӢзј©гҖӮеҚідҪҝеҜ№еҠ еҜҶж–Үд»¶еҒҡдәҶжңҖеҫ®е°Ҹзҡ„ж”№еҠЁпјҢgit д№ҹеҫ—еӯҳеӮЁж”№еҠЁеҗҺзҡ„ж•ҙдёӘж–Үд»¶пјҢиҖҢдёҚжҳҜд»…д»…еӯҳеӮЁж”№еҠЁйғЁеҲҶгҖӮ

иҷҪз„¶ git-crypt дҪҝз”Ё SHA-1 HMAC дҝқжҠӨеҚ•дёӘж–Үд»¶еҶ…е®№пјҢдҪҶеҰӮжһңж•ҙдёӘд»“еә“жІЎжңүеҒҡйҳІзҜЎж”№дҝқжҠӨпјҢgit-crypt е°ұдёҚиғҪе®үе…ЁдҪҝз”ЁпјҲеҰӮжһңж”»еҮ»иҖ…еҸҜд»ҘзҜЎж”№дҪ зҡ„д»“еә“пјҢд»–д»¬е°ұеҸҜд»Ҙдҝ®ж”№дҪ зҡ„. gitattributes ж–Үд»¶жқҘзҰҒз”ЁеҠ еҜҶпјүгҖӮеҰӮжһңеҝ…иҰҒпјҢеҸҜд»ҘдҪҝз”Ё git зҡ„зү№жҖ§пјҢдҫӢеҰӮзӯҫеҗҚж ҮзӯҫпјҢиҖҢдёҚжҳҜд»…д»…дҫқйқ  git-crypt жқҘдҝқиҜҒе®Ңж•ҙжҖ§гҖӮ

дҪҝз”Ё git-crypt еҠ еҜҶзҡ„ж–Үд»¶дёҚиғҪз”Ё git-apply жү“иЎҘдёҒпјҢйҷӨйқһиЎҘдёҒжң¬иә«д№ҹжҳҜеҠ еҜҶзҡ„гҖӮиҰҒз”ҹжҲҗдёҖдёӘеҠ еҜҶзҡ„иЎҘдёҒпјҢеҸҜд»ҘдҪҝз”Ёе‘Ҫд»Ө git diff --no-textconv --binary вҖӢгҖӮжҲ–иҖ…пјҢдҪ д№ҹеҸҜд»ҘеңЁ git еӨ–йғЁдҪҝз”Ё patch е‘Ҫд»ӨжқҘжү“жҳҺж–ҮиЎҘдёҒгҖӮ

.gitattributes ж–Үд»¶зҡ„ж–ҮжЎЈдҪҚдәҺ gitattributes (5) жүӢеҶҢйЎөйқўдёӯгҖӮж–Үд»¶еҢ№й…ҚжЁЎејҸдёҺ. gitignore дёӯдҪҝз”Ёзҡ„ж јејҸзӣёеҗҢпјҢиҜҰжғ…еҸҜеҸӮи§Ғ gitignore (5) жүӢеҶҢйЎөйқўпјҢдҪҶйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢд»…жҢҮе®ҡдёҖдёӘзӣ®еҪ•пјҲдҫӢеҰӮ /dir/ вҖӢпјүдёҚи¶ід»ҘеҜ№иҜҘзӣ®еҪ•дёӢзҡ„жүҖжңүж–Үд»¶иҝӣиЎҢеҠ еҜҶгҖӮ

еҸҰиҜ·жіЁж„ҸпјҢdir/* вҖӢжЁЎејҸдёҚдјҡеҢ№й…Қ dir вҖӢеӯҗзӣ®еҪ•дёӢзҡ„ж–Үд»¶гҖӮеҰӮжһңиҰҒеҠ еҜҶж•ҙдёӘ dir вҖӢеӯҗж ‘пјҢиҜ·дҪҝз”Ё dir/** вҖӢпјҡ

dir/** filter=git-crypt diff=git-crypt
1

.gitattributes ж–Үд»¶дёҚиғҪиў«еҠ еҜҶпјҢеӣ жӯӨиҜ·зЎ®дҝқйҖҡй…Қз¬ҰдёҚдјҡж„ҸеӨ–ең°еҢ№й…ҚеҲ°е®ғгҖӮеҰӮжңүеҝ…иҰҒпјҢдҪ еҸҜд»ҘеғҸиҝҷж ·е°Ҷ .gitattributes жҺ’йҷӨеңЁеҠ еҜҶд№ӢеӨ–пјҡ

.gitattributes !filter !diff
