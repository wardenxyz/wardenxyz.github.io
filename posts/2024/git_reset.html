git 重置当前分支¶

git reset 命令用于重置当前分支的 HEAD 到指定的状态。以下是一些常用的 git reset 命令：

1. 软重置（保留工作目录和暂存区的更改）：¶

git reset --soft

2. 混合重置（保留工作目录的更改，但重置暂存区）：¶

git reset --mixed

这是默认选项，如果不指定参数，git reset 会默认使用 --mixed。

3. 硬重置（丢弃工作目录和暂存区的更改）：¶

git reset --hard

其中 是你想重置到的提交的哈希值。如果你想重置到上一个提交，可以使用 HEAD^ 代替 。例如：

git reset --hard HEAD^
