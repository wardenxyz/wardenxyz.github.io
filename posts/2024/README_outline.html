Foobar （项目名称）¶

Foobar is a Python library for dealing with word pluralization.（项目简介）

Installation（安装）¶

Use the package manager pip to install foobar.

pip install foobar

Usage（使用）¶

import foobar
## returns 'words'
foobar.pluralize('word')
## returns 'geese'
foobar.pluralize('goose')
## returns 'phenomenon'
foobar.singularize('phenomena')
